 SAARTEXT.de als Progressive Web App    
                                        
 Den SAARTEXT nutzen, wie mit einer     
 vollwertigen App - das ist jetzt mög-  
 lich. Die Webseite des SAARTEXTes      
 (www.saartext.de) steht jetzt als Pro- 
 gressive Web App (PWA) zur Verfügung.  
                                        
 Sie kann unter Android, Apple und mit  
 dem Google-Chrome-Browser auf allen un-
 terstützen Endgeräten installiert wer- 
 den. Das mobile Abrufen der Meldungen  
 funktioniert damit schneller.          
                                        
 PWAs funktionieren für den Nutzer ähn- 
 lich wie "richtige Apps". Nach der In- 
 stallation wird ein App-Icon auf den   
 Desktops der Endgeräte angezeigt. Die  
 App kann dann gestartet werden.
