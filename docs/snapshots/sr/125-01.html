                            NACHRICHTEN 
                               Bierfeld 
                                        
 Ruhestörung durch Autofahrer           
                                        
 Die Polizei ermittelt gegen einen 51-  
 Jährigen, der in der Nacht zum Freitag 
 eine Ruhestörung durch einen aufheulen-
 den Motor verursacht haben soll. Eine  
 Anwohnerin hatte sie verständigt.      
                                        
 Ein Atemalkoholtest bestätigte, dass   
 der Beschuldigte betrunken war. Sein   
 Auto war weder angemeldet noch haft-   
 pflichtversichert. Er musste seinen    
 Führerschein sofort abgeben.           
                                        
 Eine 42-Jährige, die den Autofahrer    
 kennt, beleidigte die Beamten, als sie 
 den Vorfall aufnahmen. Deshalb wurde   
 gegen sie eine Strafanzeige wegen des  
 Verdachts auf Beleidigung gestellt.
