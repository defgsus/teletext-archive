                         CORONAPANDEMIE 
                          Coronaimpfung 
                                        
 Wie bekomme ich einen Impftermin?      
                                        
 Termine für eine Coronaschutzimpfung in
 einem der vier saarländischen Impfzen- 
 tren können online oder telefonisch ge-
 bucht werden. Online-Buchungen sind un-
 ter www.impfen-saarland.de möglich.    
                                        
 Auf dem Buchungsportal muss erst ausge-
 wählt werden, ob es sich um eine Erst-,
 Zweit- oder Boosterimpfung oder eine   
 Impfung für Minderjährige handelt. Dann
 kann das Impfzentrum gewählt werden.   
                                        
 Alle verfügbaren Termine werden dann   
 angezeigt und können ausgewählt werden.
 Auch eine telefonische Terminvergabe   
 ist möglich unter 0681/502-4422 oder   
 unter 0800/9991599.
