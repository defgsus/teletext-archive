                   Aufschrei der Jugend 
                                        
 Generation "Fridays for Future"        
                                        
 Seit März 2019 begleitet Kathrin Pit-  
 terling Berliner Protagonisten von     
 "Fridays for Future". Die Doku zeigt,  
 wie vielfältig, aber auch kräftezehrend
 Protestarbeit ist.                     
                                        
 Die Proteste bringen viele an ihre phy-
 sischen und psychischen Grenzen. Einer-
 seits erleben sie Bestätigung, z.B.als 
 sie 2019 für eine Demo in Berlin       
 250 000 Menschen mobilisieren konnten. 
                                        
 Andererseits überfällt sie zunehmend   
 ein Ohnmachtsgefühl, als sie merken,   
 dass sie politisch weit weniger bewegen
 als erhofft. Können sie mit ihrem      
 friedlichen Protest etwas ausrichten?
