                           SR FERNSEHEN 
                     Mainz bleibt Mainz 
                                        
 Die frühen Jahrzehnte - Das Beste      
                                        
 2015 feierte die Mainzer Fernsehfast-  
 nacht 60.Geburtstag. In einer dreitei- 
 ligen Dokumentation erinnerte der SWR  
 an die Höhepunkte einer einmaligen     
 Erfolgsgeschichte.                     
                                        
 Teil eins widmet sich den Jahren, in   
 denen legendäre Auftritte von Mainzer  
 Ikonen die Sendung in Deutschland zum  
 Straßenfeger machten. Die Rede ist von 
 Ernst Neger oder Margit Sponheimer.    
                                        
 Dazu gibt es ein Wiedersehen mit den   
 besten Auftritten vieler Vorzeigekünst-
 ler der Mainzer Fernsehfastnacht. Er-  
 gänz wird das ganze durch sachkundige  
 Kommentare von Zeitzeugen.
