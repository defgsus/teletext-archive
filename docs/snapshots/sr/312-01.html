                           SR FERNSEHEN 
                      Um Himmels Willen 
                                        
 Schuldgefühle                          
                                        
 Schwester Hanna findet heraus, dass    
 Bürgermeister Wöller eine Kaserne nebst
 Truppenübungsplatz auf dem Klosterge-  
 lände errichten will. Sie gründet eine 
 Bürgerinitiative.                      
                                        
 Die Arbeiten laufen auf Hochtouren. Es 
 werden Flugblätter verteilt und eine   
 Online-Petition geplant. Natürlich ist 
 Wöller ganz und gar nicht erfreut, dass
 Hanna so viel Unruhe stiften will.     
                                        
 Denn noch hat sich das Ministerium     
 nicht für Kaltenthal als Standort ent- 
 schieden. Wenn Hanna damit an die Öf-  
 fentlichkeit geht, könnte sie das ganze
 Projekt gefährden.                     
      Samstag, 29.Januar, 11.45 Uhr
