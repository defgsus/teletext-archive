Das Erste: Neues 
Zwei neue Folgen von "Der Ranger"      
                                       
Eine Untersuchung bringt eine erschre- 
ckende Diagnose, die nicht nur Lauras  
Beziehung zu Florian schwer belastet,  
sondern auch Lauras Exfreund Alex      
(Joscha Kiefer) auf den Plan ruft ...  
                                       
Zum Inhalt von "Zusammenhalt": Marie   
Fechner (Anna Herrmann), die alleine   
mit drei jüngeren Geschwistern auf dem 
Nachbarhof der Waldeks wohnt, rettet   
einen durch ein Windrad verletzten Rot-
milan. Nun soll unter Beteiligung von  
Karl Nollau ein weiterer Windpark in   
unmittelbarer Nähe entstehen - eine Be-
drohung für die Natur, die Jonas und   
Marie unbedingt verhindern wollen.
