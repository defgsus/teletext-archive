Verbraucher: Nachrichten 
Urteil zur Witwenrente gesprochen      
                                       
Für eine betriebliche Hinterbliebenen- 
versorgung dürfen Arbeitgeber eine Min-
destdauer der Ehe festlegen. Dabei ist 
eine Frist von einem Jahr, mit bestimm-
ten Ausnahmen, "noch angemessen", wie  
das Bundesarbeitsgericht entschied.    
                                       
                                       
Die Klägerin hatte 2018 geheiratet. Ihr
Mann starb vier Monate später. Den An- 
trag der Witwe auf Hinterbliebenenver- 
sorgung lehnte der Arbeitgeber unter   
Hinweis auf die Versorgungsregelungen  
des Unternehmens ab. Sie schließen eine
Witwenrente aus, wenn die Ehe keine 12 
Monate bestand. (AZ: 3 AZR 254/21)
