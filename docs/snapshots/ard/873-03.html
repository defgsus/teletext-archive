Bundestag redet über Impf-Pflicht      
                                       
Deshalb wurde im Bundestag auch über   
eine Pflicht diskutiert, sich beraten  
zu lassen.                             
                                       
Eine Impf-Pflicht bedeutet nicht Impf- 
Zwang. Niemand wird gegen seinen Willen
geimpft. Aber man muss dann zum        
Beispiel eine Strafe zahlen.
