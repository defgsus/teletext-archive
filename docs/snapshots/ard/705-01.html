Lagarde schließt Zinserhöhung nicht    
mehr ganz aus

kleine Hinweise auf eine nahende Zins- 
wende im Euroraum.

terhin gelte, dass es dieses Jahr sehr 
wahrscheinlich keine Zinserhöhungen ge-
ben werde, antwortete Lagarde, dass    
dies von den Daten abhänge, überstürzen
wolle man nichts.
