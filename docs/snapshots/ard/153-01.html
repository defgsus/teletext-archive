Lindner bringt verlängerte Steuerhilfen
wegen Corona-Pandemie auf den Weg      
03.02.2022 14:03                       
                                       
Wegen der anhaltenden Corona-Pandemie  
hat Bundesfinanzminister Christian     
Lindner (FDP) eine Verlängerung von    
Steuererleichterungen auf den Weg ge-  
bracht. Geplant sei die Beibehaltung   
der Homeoffice-Pauschale für Arbeitneh-
mer, die von zuhause aus arbeiten,     
schrieb Lindner auf Twitter. Zudem soll
demnach erneut eine spätere Abgabefrist
für die Steuererklärung gelten....
