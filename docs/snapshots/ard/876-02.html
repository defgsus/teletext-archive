Extra: Nachrichtenleicht  
Microsoft kauft Spiele-Entwickler      
                                       
Das ist englisch und heißt auf deutsch:
"Die Pflicht ruft". Ein anderes Spiel  
ist "World of Warcraft". Das heißt:    
"Die Welt von dem Kriegs-Handwerk".    
                                       
Microsoft will für Activision-Blizzard 
fast 70 Milliarden Dollar bezahlen. Das
wäre der höchste Preis für ein         
Computer-Spiel-Unternehmen. Microsoft  
darf aber nur kaufen, wenn die Behörden
in den USA es erlauben. Das wird jetzt 
geprüft.
