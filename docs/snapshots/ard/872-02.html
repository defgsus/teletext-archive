Extra: Nachrichtenleicht  
So viele Neu-Infektionen wie nie       
                                       
Die Experten warnen: Es gibt vielleicht
viel mehr Corona-Fälle. Ob man sich    
angesteckt hat, wird durch einen Test  
nachgewiesen. Die Test-Zentren kommen  
aber mit dem Auswerten von den Tests   
kaum nach, weil sie so viel Arbeit     
haben.                                 
                                       
Bundes-Kanzler Olaf Scholz und die     
Minister-Präsidentinnen und Minister-  
Präsidenten haben wegen der hohen      
Zahlen beraten, wie es weitergeht. Sie 
haben entschieden: Die Corona-Schutz-  
Maßnahmen bleiben so, wie sie sind.
