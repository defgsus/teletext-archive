Großes Interesse an Ökolandbau         
                                       
Das Interesse an einer Umstellung auf  
Ökolandbau ist nach Angaben des Deut-  
schen Bauernverbandes (DBV) so groß wie
nie. Laut einer Umfrage unter 1500 Be- 
trieben sind 20 % an einer Öko-Umstel- 
lung interessiert, teilte der DBV mit. 
18,7 % könnten sich eine Umstellung    
vorstellen, 1,3 % planen sie bereits   
fest ein.                              
                                       
"In Süddeutschland sind sogar 27,5 %   
der Betriebe offen für eine Umstel-    
lung", erklärte der DBV weiter. Beson- 
ders kleinere Betriebe unter 50 Hektar 
interessieren sich demnach für den Öko-
landbau.
