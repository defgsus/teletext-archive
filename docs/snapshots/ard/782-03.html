Fernsehen: Technik 
Manueller Sendersuchlauf               
                                       
Wenn Sie die neuen HD-Sender gezielt   
suchen wollen, müssen Sie im Sender-   
suchlauf-Menü (Installations-Menü)     
technische Parameter der neu genutzten 
Satelliten-Transponder, wie z.B. Fre-  
quenz, Symbolrate und Polarisation     
eingeben.                              
                                       
Ist der gewünschte Transponder nicht   
auswählbar, können Sie diesen neu an-  
legen, entsprechende Menü-Einträge     
lauten z.B. "benutzerdefiniert" oder   
"neu". Eine genaue Beschreibung finden 
Sie in der Bedienungsanleitung Ihres   
Receivers oder auf den Internetseiten  
der Hersteller.
