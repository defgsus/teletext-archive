 Weiterer Antrag auf Impfpflicht        
                                        
 Zur Einführung einer Corona-Impfpflicht
 liegt jetzt ein weiterer Gesetzentwurf 
 vor.                                   
                                        
 Eine Gruppe um den FDP-Bundestags-Abge-
 ordneten Ulmann plädiert für eine Impf-
 pflicht für Menschen ab 50 Jahren. Sie 
 soll aber nur eingeführt werden, wenn  
 dies aufgrund der Pandemie-Situation im
 kommenden Herbst für notwendig erachtet
 wird. Zuvor soll verpflichtend beraten 
 werden.                                
                                        
 Der Entwurf wird - ebenso wie die ande-
 ren vorliegenden - im März im Bundestag
 beraten.
