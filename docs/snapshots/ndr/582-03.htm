 SO BLEIBT DER TOD BEZAHLBAR (3/11)     
                                        
 Grundsätzlich müssen Tote in Deutsch-  
 land auf dafür zugelassenen Flächen    
 beigesetzt werden, meist auf Friedhö-  
 fen, aber auch zunehmend unter Bäumen  
 auf speziellen Waldflächen. Einzige    
 Ausnahme: In Bremen darf die Asche ei- 
 nes Verstorbenen unter zahlreichen Be- 
 dingungen seit 2015 auch auf privaten  
 Grundstücken verstreut werden. Urnen   
 müssen auch dort weiterhin auf Friedhö-
 fen beigesetzt werden.                 
                                        
 Friedhöfe werden in der Regel von Ge-  
 meinden betrieben, vielfach aber auch  
 von Kirchengemeinden. Eine Alternative 
 sind Seebestattungen.
