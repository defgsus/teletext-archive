 PREMIUMGEHÖLZE, GETESTET & PRÄMIERT    
                                        
 Geachtet wird dabei vor allem auf den  
 Zierwert, die Wüchsigkeit, die Frost-  
 härte und das Blühverhalten. Wichtig   
 ist zusätzlich, ob die Gehölze anfällig
 für Krankheiten oder Schädlinge sind.  
                                        
 Die Bewertungsskala reicht von 0 bis 3 
 Sternen. Nur die Gehölze, die mit drei 
 Sternen bewertet werden, dürfen das    
 Prädikat "Premiumgehölz der Bundesge-  
 hölzsichtung" tragen. Zu erkennen sind 
 sie an einem kleinen Schild, auf dem   
 ein stilisiertes, goldenes Eichenblatt 
 abgebildet ist.
