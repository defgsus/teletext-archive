    Untertitelung im NDR Text   
 Zu Beginn einer Sendung wird ein Symbol
 eingeblendet, das auf die Untertitelung
 hinweist. Eine Übersicht der im NDR un-
 tertitelten Sendungen finden Sie auf   
 Seite 562.                             
                                        
 Es gibt mehrere Möglichkeiten, die Un- 
 tertitel zu empfangen. Um die klassi-  
 schen Videotext-Untertitel zu aktivie- 
 ren, müssen Sie im NDR Text die Seite  
 150 aufrufen.                          
                                        
 Die Untertitel können auch digital über
 den Verbreitungsweg DVB empfangen wer- 
 den. Dafür muss die entsprechende Funk-
 tion an dem TV-Gerät oder dem Empfangs-
 gerät aktiviert werden. Meist gibt es  
 dafür eine Taste auf der Fernbedienung.
