 Je nach Größe des Fleischstücks und je 
 nach Ofen kann die Zubereitungszeit    
 schwanken. Bei einer Kerntemperatur von
 68 Grad ist das Fleisch saftig fertig  
 gegart.                                
                                        
 Indirekte Hitze im Grill wird erreicht,
 indem die Kohle nur auf einer Seite des
 Grills aufgeschichtet wird. Das Fleisch
 kommt dann auf die Seite ohne Kohle.   
 Beim Grillen kann der Geschmack noch   
 verstärkt werden, indem man auf die    
 Glut etwas Hickory-Holz gibt. Dadurch  
 bekommt das Fleisch ein zusätzliches   
 Räucheraroma.
