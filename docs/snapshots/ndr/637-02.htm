    09.02.2022                  
                                        
 Problem is, dat se bi ehr Arbeid jo ok 
 nich bloots op ene Stell steiht.       
                                        
 Nee, se mutt vun Disch na 'n Heerd, vun
 Heerd an de Spööl, verscheden Saaken   
 ut' Schapp halen, wedder an Disch tor- 
 üch. Rümflitzen as de Huusfru bi Lori- 
 ot, de ehrn Hermann dat Rutgohn smack- 
 haft maken will.                       
                                        
 Un so loopt wi uns bi't Diskereern in  
 de Köök ümmer vör de Fööt. Se schimpt: 
 "Wat löppst du mi ümmerto in Weg!      
                                        
 Beweeg di doch maal dor, woans ik      
 bloots selten hen mutt!" Kunnen wi uns 
 aver nich op eenigen.
