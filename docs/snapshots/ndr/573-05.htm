 HAUSFASSADE RICHTIG STREICHEN (5/10)   
                                        
 Grundierung muss dann auf die Fassade, 
 wenn beim Wischen über den Putz Krümel 
 an der Hand hängenbleiben oder wenn die
 Fassade die Feuchtigkeit wie ein       
 Schwamm aufsaugt - so wie etwa ältere  
 Klinkersteine. Für jede Fassadenober-  
 fläche gibt es die passende Grundie-   
 rung. Sie wird mit einer Tiefengrund-  
 bürste kreuzförmig aufgetragen.        
                                        
 All diese Vorbereitungen dauern insge- 
 samt mehrere Tage. Denn die Fassade    
 muss für das Ausbessern, die Vorbehand-
 lung gegen Algen und Moose sowie die   
 Grundierung immer ganz trocken sein.
