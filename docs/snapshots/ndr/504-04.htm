 Die Navigation von Seite zu Seite ist  
 bei modernen Teletext-Dekodern auch    
 über die Pfeil-Tasten der Fernbedienung
 möglich. Dreiziffrige Zahlen werden so 
 direkt vorgewählt, mit der "OK"-Taste  
 die Seiten unmittelbar aufgerufen.     
                                        
 Inhalt (A-Z), eine Übersicht über alle 
 NDR Text-Seiten, alphabetisch sortiert,
 findet sich auf den Seiten 101 bis 107.
                                        
 Unter www.ndrtext.de finden Sie den NDR
 Text auch im Internet. Hier steht Ihnen
 auch eine aktuelle Seitenübersicht zum 
 Download zur Verfügung.
