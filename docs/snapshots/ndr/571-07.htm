 E-AUTO GEBRAUCHT KAUFEN: CHECKLISTE    
                                        
 Grund hierfür: Der Wertverlust kann im 
 ersten oder sogar zweiten Jahr niedri- 
 ger als die Kaufprämie ausfallen.      
                                        
 Beispiel: Ein E-Auto kostet neu 39.000 
 Euro. Durch die Kaufprämie von insge-  
 samt 9.000 Euro sinkt der Anschaffungs-
 preis auf 30.000 Euro. Verliert das Au-
 to im ersten Jahr aber nur 6.000 Euro  
 an Wert, lässt es sich immer noch für  
 33.000 Euro verkaufen. Unterm Strich   
 bleibt damit sogar ein Gewinn von 3.000
 Euro.
