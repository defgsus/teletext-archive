 ERKENNEN SCHNELLTESTS OMIKRON? (2/7)   
                                        
 So kommt es leichter vor, dass zumin-  
 dest einige Schnelltests auch hoch po- 
 sitive, infektiöse Betroffene überse-  
 hen. Das Paul-Ehrlich-Institut (PEI)   
 untersucht zurzeit, welche Tests auch  
 bei Omikron zuverlässig sind. Ergebnis-
 se sollen Mitte Februar vorliegen.     
                                        
 Viele Tests reagieren erst, wenn sehr  
 viele Viren im Rachen vorhanden sind.  
 Deshalb ist das Timing beim Testen ent-
 scheidend: Zwischen einer niedrigen und
 einer sehr hohen Viruslast können bei  
 Omikron nur wenige Stunden liegen, weil
 es sich im Rachenraum viel schneller   
 vermehrt als vorherige Varianten.
