 Buxtehuder SV: Umbruch im Sommer       
                                        
 Im Kader des Handball-Bundesligisten   
 Buxtehuder SV wird es zur kommenden    
 Saison einen Umbruch geben.            
                                        
 Vier Leistungsträgerinnen verlassen den
 Verein, eine neue Torfrau kommt. Das   
 gab der BSV am Montag bekannt.         
                                        
 National-Torhüterin Katharina Filter   
 will sich eine neue sportliche Heraus- 
 forderung suchen, heißt es in der Mit- 
 teilung. Nachfolgerin wird Marie Andre-
 sen von der HSG Blomberg-Lippe. Zudem  
 gehen: Rückraumspielerin Annika Lott,  
 Kreisläuferin Lisa Antl und Rechtsaußen
 Meret Ossenkopp.
