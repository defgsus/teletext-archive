 Windstärke 10: Schwerer Sturm          
 (89-102 km/h)                          
                                        
 Bäume werden entwurzelt, Stämme brechen
 ab. An Häusern können größere Schäden  
 entstehen. Der Seegang zeichnet sich   
 durch sehr hohe Wellen aus - die schwe-
 ren Brecher.                           
                                        
 Windstärke 11: Orkanartiger Sturm      
 (103-117 km/h)                         
                                        
 Die See "brüllt". Sehr hohe Wellenberge
 entstehen, das Wasser wird fast waage- 
 recht weggeweht. An Land werden Dächer 
 abgedeckt, Autos umgeworfen, in Wäldern
 richtet Windbruch schwere Schäden an.
