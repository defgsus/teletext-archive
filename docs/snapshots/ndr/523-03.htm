 OMIKRON: SITUATION IN KLINIKEN (3/7)   
                                        
 Bei jedem Kontakt muss das Pflegeperso-
 nal Schutzkleidung anlegen. Jede Pfle- 
 gemaßnahme ist deshalb viel aufwendiger
 und zeitraubender als auf einer norma- 
 len Station. So dauert ein Verbands-   
 wechsel statt einer Viertelstunde oft  
 doppelt so lange.                      
                                        
 Ein Problem gab es schon vor Corona:   
 Den Kliniken fehlt Pflegepersonal. Zur-
 zeit fallen zusätzlich Mitarbeitende   
 aus, weil sie in Quarantäne oder selbst
 infiziert sind.
