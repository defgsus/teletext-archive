 Coronatallen                           
                                        
 Dat Robert-Koch-Institut mellt vunmor- 
 gen goot 236.000 ne'e Coronainfekscho- 
 nen in Düütschland. De Inzidenz klat-  
 tert op 1.283.                         
                                        
 Ok wenn de Tallen jüst so hooch sünd,  
 see Bunnsjustizminister Marco Busch-   
 mann, een kunn dor bald över snacken,  
 wat för Regeln se in Düütschland lo-   
 ckern köönt. Wenn de Gefohr för de     
 Sellschop nich mehr so hooch is, denn  
 mööt ok de Oplangen wedder trüchnahmen 
 warrn, meen de FDP-Politiker.
