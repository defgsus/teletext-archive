 Bovenschulte weiter für Impfpflicht    
                                        
 Bremens Bürgermeister Andreas          
 Bovenschulte (SPD) hat sich im Inter-  
 view auf NDR Info am Mittwoch weiterhin
 für die Einführung einer allgemeinen   
 Corona-Impfpflicht ausgesprochen.      
                                        
 Es müsse eine "Grundimmunität von      
 deutlich über 90 Prozent" erreicht wer-
 den. "Ich bin sehr dafür, dass das über
 das Impfen erreicht wird, weil damit   
 weitaus weniger Kosten verbunden sind  
 und weitaus weniger Leid", sagte er.   
                                        
 Wenn man in Deutschland flächendeckend 
 eine so hohe Quote erreichen wolle, sei
 dies nur über eine Impfpflicht möglich.
