 Güng ik an't Finster lang hett se mi   
 püstert, drück ik mi an Disch vörbi, se
 glieks so mit'n Handrügg an't Weddeln. 
 Bedüüd "Weg dor!"                      
                                        
 Na, denk ik, dat lött sik wiss organi- 
 seern. Ik warr wull rutkriegen of för  
 mi Wegen överblievt. Heff mien Schatz  
 'n poor Daag heemlich beluert un rut-  
 funnen, wölk Strecken se in ehr Köök   
 raar lööp, op wölk Fliesen se nie pett.
                                        
 Allens op'n Zeddel fastholen. As se in-
 köpen weer, heff ik all düsse Footbo-  
 ddenkacheln mit bunte Opkleber för mi  
 kenntekent. Wat passeert? Groot Theo-  
 ter.
