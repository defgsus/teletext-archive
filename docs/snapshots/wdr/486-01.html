      Einstellhilfe             

                                        
 

Einstellhilfe für DVB-S-Receiver       

                                        
 Wenn Sie nach der Sendung Lokalzeit    
 nicht mehr das WDR-Fernsehen empfangen,
 müssen Sie lediglich eine Kleinigkeit  
 beachten.                              
                                        
 Wechseln Sie einfach kurz auf ein      
 anderes Programm und kehren dann wieder
 zurück auf den Kanal, auf dem Sie WDR  
 Fernsehen programmiert haben.          
                                        
 Eine bessere Lösung bietet ein Soft-   
 ware-Update, um das Problem zu be-     
 seitigen:
